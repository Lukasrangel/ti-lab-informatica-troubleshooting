# Troubleshooting de Laboratório de Informática

Este repositório documenta problemas técnicos comuns encontrados na manutenção de um laboratório de informática utilizado para aulas de programação.

O objetivo é registrar processos de diagnóstico e solução para ambientes de desenvolvimento, sistemas operacionais e ferramentas utilizadas em sala de aula.

## Conteúdos

- Configuração de ambientes de desenvolvimento
- Conflitos de portas
- Problemas comuns em Windows
- Preparação de máquinas para aula

## Objetivo

Criar uma base de conhecimento para troubleshooting e documentação técnica de suporte em ambientes de laboratório.
